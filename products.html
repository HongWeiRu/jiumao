產品介紹

專業電梯製造，滿足各種需求

電梯類型

乘客電梯

適用於住宅大樓、商業大樓、辦公大樓等場所，提供舒適安全的乘載服務。 設計符合人體工學，運行平穩安靜，讓您的每一次搭乘都成為舒適的體驗。

  • ✓ 載重：630kg ~ 1600kg
  • ✓ 速度：60m/min ~ 105m/min
  • ✓ 適用於各類建築物

醫用電梯

專為醫療院所設計，符合醫療環境特殊需求。 具備寬敞的車廂空間，方便病床及醫療設備進出，確保醫療服務的順暢進行。

  • ✓ 寬敞車廂設計
  • ✓ 符合醫療環境規範
  • ✓ 平穩運行，保護病患安全

客貨電梯

兼具載客與載貨功能，適合商業及工業用途。 堅固耐用的設計，可同時滿足人員與貨物的運輸需求，提升使用效率。

  • ✓ 載重：1000kg ~ 2000kg
  • ✓ 多功能設計
  • ✓ 適合商業及工業用途

油壓電梯

適合低層建築使用，運行平穩可靠。 採用油壓系統驅動，結構簡單、維護方便，是低層建築的理想選擇。

  • ✓ 適合低層建築
  • ✓ 運行平穩
  • ✓ 維護方便

停車電梯

專為停車場設計，提升空間利用效率。 可將車輛垂直運送至不同樓層，有效解決停車空間不足的問題。

  • ✓ 提升空間利用效率
  • ✓ 專為車輛設計
  • ✓ 解決停車空間問題

工廠電梯

工業級電梯，適合工廠及倉儲使用。 堅固耐用的設計，可承受重載及頻繁使用，滿足工業環境的嚴苛需求。

  • ✓ 工業級設計
  • ✓ 重載能力
  • ✓ 適合工廠及倉儲

送菜梯

專為餐廳、飯店設計，方便食材運送。 小巧的設計適合廚房環境，可快速將食材運送至不同樓層，提升工作效率。

  • ✓ 小巧設計
  • ✓ 適合廚房環境
  • ✓ 提升工作效率
